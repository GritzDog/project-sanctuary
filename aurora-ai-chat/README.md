## üöÄ Getting Started

### Prerequisites
- Node.js and npm
- Rust and Cargo
- Ubuntu 24.04+ (tested) or compatible Linux distribution

### Installation & Running

1. **Install dependencies**:
```bash
npm install
```

2. **Start the native desktop app**:
```bash
npm run tauri dev
```

3. **Build for production**:
```bash
npm run tauri build
```

4. **Start chat interface only**:
```bash
npm run chat
```
